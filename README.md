* STM32CubeMX Version 4.17.0
* HAL Libraries - FW_F4 V1.13.1
* Keil uVision 5.21a
* EagleCAD 7.2
* Autodesk Inventor 2016

Development tools can downloaded [here](https://drive.google.com/open?id=0B6SlddUSFtelRTBFR0EwWk1qT1k)
